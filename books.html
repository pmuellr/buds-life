Maurice "Bud" Mueller, MD, PhD, wrote an autobiography titled "Bud's Life". It's available in several formats:

  • Paperback 6"x9" from Lulu for $8.40
  • in the Amazon Kindle store for $0.99
  • PDF format for 8.5"x 11" pages
  • EPUB format for eReaders except the Kindle
  • HTML format to read in a web browser
  • MOBI format for the Kindle (for computer nerds)

If you have an new-ish iPod Touch, iPhone, or iPad you can download the EPUB version, drop in your Books folder in iTunes, and read it with the iBooks application on your device.
